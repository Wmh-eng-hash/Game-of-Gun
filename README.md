# 自动打怪 · 突破增幅（移动版）

一款轻量级 Roguelike 挂机战斗游戏，专为移动端优化。您操控角色在塔中不断向上攀登，自动攻击怪物，并通过“突破增幅”系统强化自身。支持横屏/竖屏自适应，操作参考主流手游（左下摇杆 + 右下攻击键）。

## ✨ 功能特性

- **自动战斗**：角色会自动攻击最近的怪物，子弹自动追踪。
- **手动攻击**：点击右下角圆形按钮可立即发射一枚子弹（不受冷却限制）。
- **突破增幅**：消耗金币升级生命、攻击、防御、攻速、暴击，数值呈指数增长，突破阈值后获得巨额加成。
- **楼层推进**：每层清空怪物或倒计时结束自动进入下一层，怪物强度随层数指数上升（每 5 层 Boss 战）。
- **死亡惩罚**：生命归零后退回上一层，并恢复 50% 生命值。
- **离线存档**：游戏进度自动保存在浏览器本地（支持校验防篡改）。
- **全屏适配**：完美支持手机横屏/竖屏，兼容刘海屏安全区域。

## 🎮 操作指南

| 操作 | 说明 |
|------|------|
| **移动** | 触摸屏幕左下区域（摇杆热区）并拖拽；或使用键盘方向键（PC 调试）。 |
| **手动攻击** | 点击右下角圆形 ⚔️ 按钮，立即发射一颗子弹。 |
| **升级** | 点击底部面板中的升级按钮（生命/攻击/防御/攻速/暴击），消耗对应金币。 |
| **自动攻击** | 角色会按当前攻速自动攻击最近的怪物。 |

> 提示：金币通过攻击怪物获得，击败怪物额外奖励金币。

## 🛠 技术栈

- 原生 HTML5 + CSS3 + JavaScript (ES6)
- Canvas 2D 渲染
- localStorage 数据持久化
- 无任何第三方库，完全自包含单文件

## 📱 移动适配亮点

- 响应式布局，横屏/竖屏自动调整控件大小
- 摇杆热区扩大至屏幕左下 40% 区域，操作更顺手
- 攻击按钮 70px 大触控区域，符合 Apple HIG 和 Material Design 标准
- 顶部状态栏半透明毛玻璃效果，不遮挡视野
- 底部升级面板横向滚动，适配小屏
- 支持 `env(safe-area-inset-*)`，全面屏无遮挡

## 💾 存档与安全

- 自动保存于浏览器 `localStorage`，每 5 秒或关键事件触发保存。
- 存档包含校验和（Checksum），防止恶意篡改；若校验失败，游戏自动重置。
- 旧版本存档自动兼容补全。

## 🧪 测试环境

- 浏览器：Chrome / Safari / Edge 最新版
- 系统：Android 10+ / iOS 14+
- 推荐使用手机 Chrome 或 Safari 全屏模式获得最佳体验

## 📦 部署方式

直接运行 `index.html` 文件即可，所有资源内联。若需部署至 Web 服务器，仅需托管该单一文件。
